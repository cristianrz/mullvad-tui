# Mullvad T-UI

Two terminal user interfaces:

* Wizard
* Configuration

## Getting Started

### Prerequisites

* [Mullvad CLI app](https://mullvad.net/en/download/)
* `git`
* `make`

### Installing

Download the repository

```
git clone https://github.com/cristianrz/mullvad-tui.git
```

and install

```
make install
```

this will give you two commands:

* `mullvad-wizard`
* `mullvad-configure`
